# Contracts

Foundry workspace untuk `Last Nads Standing`.

## Included

- `src/`: `GameFactory.sol`, `GameRoom.sol`, `PredictionPool.sol`
- `script/`: deploy dan interact script
- `test/`: unit test dan integration test
- `deployments/`: ABI dan address deploy Monad testnet

## Notes

- Dependency contract tidak di-commit dari repo sumber build outputnya.
- Install library Foundry yang dibutuhkan sebelum build jika folder `lib/` belum ada.
